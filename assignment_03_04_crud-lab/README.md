# Assignment 03/04/2026 - CRUD Lab

Implement CRUD operations using Mongoose.

## Setup

```bash
npm install
npm start
```

## API

- `GET /api/tasks`
- `GET /api/tasks/:id`
- `POST /api/tasks`
- `PUT /api/tasks/:id`
- `DELETE /api/tasks/:id`

## Notes

This assignment uses a MongoDB database at `mongodb://127.0.0.1:27017/crud_lab` by default.
