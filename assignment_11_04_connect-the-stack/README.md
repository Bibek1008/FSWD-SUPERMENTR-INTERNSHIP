# Assignment 11/04/2026 - Connect the Stack

Connect a React-style frontend to backend APIs.

## Run

```bash
npm install
npm start
```

Then open `http://localhost:3006` and click **Load Tasks**.

## Backend

- `GET /api/tasks`
- `GET /api/status`

## Frontend

A static page fetches data from the backend API and renders it in the browser.
